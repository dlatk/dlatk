=====================
--include_sub_collocs
=====================
Switch
======

--include_sub_collocs

Description
===========

When a collocation such as "happy birthday" is found in the text, it counts 1 for "happy birthday", "happy", and "birthday".

Argument and Default Value
==========================

Modifies --use_collocs

Details
=======

Modifies :doc:`fwflag_use_collocs`
