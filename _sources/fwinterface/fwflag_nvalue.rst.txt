--nvalue
========
Switch
======

--nvalue

Description
===========

Turn on (TRUE) or off (FALSE) reporting of N (number of groups) in --csv, --rmatrix, etc

Argument and Default Value
==========================

Argument: True (to report N) or False (to not report N)
