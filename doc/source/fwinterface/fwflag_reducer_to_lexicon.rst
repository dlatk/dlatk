====================
--reducer_to_lexicon
====================
Switch
======

--reducer_to_lexicon table_name

Description
===========

Writes the reduction model to a specified lexicon. The lexicon MySQL table will be written to the *permaLexicon* database by default.

Argument and Default Value
==========================

MySQL table name. There is no default.

Details
=======

See the :doc:`../tutorials/tut_clustering` tutorial for details.
